# Native Bee and Nectar-Producing Plant Community Data from Iowa and Missouri (2017-2018)

[![Data DOI](https://img.shields.io/badge/Data-edi.1210.1-orange.svg)](https://portal.edirepository.org/nis/mapbrowse?scope=edi&identifier=1210&revision=1) 

This repository contains analysis of native bee and floral resource community data. These data were collected between May and August of 2017 and 2018 in the Grand River Grasslands (GRG) of Ringgold Co. Iowa and Harrison Co. Missouri. Data--for both Midwest US-native bee and nectar-producing flower communities--are archived with the [Environmental Data Initiative](https://environmentaldatainitiative.org/) (EDI). See [here](https://portal.edirepository.org/nis/mapbrowse?scope=edi&identifier=1210&revision=1) for the full dataset with metadata and citation information.
